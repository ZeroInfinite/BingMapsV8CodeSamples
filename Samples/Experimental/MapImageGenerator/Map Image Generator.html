Map Image Generator Sample Status: Not Working
Warning: This is an experimental sample that uses the map in an unsupported way. This can break at any time.
This sample shows how to generate an image of the map. Pushpin images must be hosted locally, or be dataUri's. The following elements are not rendered; navigation controls, scale bars, Infoboxes, AnimatedTileLayers, TileLayers, and most Custom Overlays. Bing Maps terms of use does not allow Ordnance Survey or Birdseye imagery to be "printed". Streetside imagery is not supported.
